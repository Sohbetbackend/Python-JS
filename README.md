# Flask_JS
Follow step by step if you are using linux:
Open your terminal
$ git clone https://github.com/Sohbetbackend/Python-JS.git
$ cd Python-JS
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt 
$ flask db migrate -m "all"
$ flask db upgrade 
$ flask run
